# Public Folder

This folder contains static assets that are served as-is.

## Usage

Files in this folder can be referenced with absolute paths starting with `/`

### Example:
- Store SVG: `public/logo.svg`
- Use in code: `<img src="/logo.svg" />` or `url('/logo.svg')`

## Recommended for:
- Logo files
- Favicons
- Large images that don't need processing
- Files referenced in HTML
